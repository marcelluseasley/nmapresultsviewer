NMAP Results Viewer (XML)

To correctly use the NMAP Results viewer, please upload your Nmap scan results in the XML format to:

/v1/nmap

One way to do this is via a curl command:

curl /v1/nmap --upload-file [scanresults.xml]

The API response will be JSON:


{
   "scan-id": "589c048e-cca7-4686-9d70-5a4bd07669f5"
}
